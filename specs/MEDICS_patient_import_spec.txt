Patient file import specs
1) Standard TAB delimted text file, one record per line, tab character separates fields
2) Preferred file name should be INSURANCEID_YYYYMMDD.csv, where INSURACEID is the source (Wellcare/Bravo/etc)
3) Contents: 
   primary-care-provider-id
   primary-care-provider-lname
   primary-care-provider-fname
   patient-insurance-id : patient is insured by..  ie: Wellcare, Bravo, etc...
   patient-insured-id
   patient-ssn
   patient-lname
   patient-mname
   patient-fname
   patient-dob : must be formatted MM-DD-YYYY
   patient-sex
   patient-street1
   patient-street2
   patient-city
   patient-state
   patient-zip
   patient-primary-phone : formatted AAAPPPNNNN
   patient-effective-date : must be formatted MM-DD-YYYY
   patient-termination-date : must be formatted MM-DD-YYYY

 4) Data need to be posted on a secure site for download monthly or sent to the npss-dropbox@tbt.org
